## 프로젝트 소개
메가박스, CGV, 롯데시네마 사이트의 크롤링한 무대인사 정보를 지도, 달력 등으로 다양하게 모아보는 토이 프로젝트입니다.

## 주요 기능
- 시네마의 무대인사 정보 사이트 바로가기
- 무대인사 정보 지도로 보기
- 무대인사 정보 달력으로 보기
